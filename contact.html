Report an Issue

If you encounter a bug or technical issue with Altimeter, please report it on our GitHub issues page. When reporting an issue, please include:

  • Device type and operating system (e.g., iPhone 14, iOS 16.5)
  • Browser name and version (e.g., Safari 16.5, Chrome 115)
  • Steps to reproduce the issue
  • Expected behavior vs. actual behavior
  • Any error messages displayed
  • Screenshot if applicable

Feature Requests

Have an idea for a new feature or improvement? We're always looking to make Altimeter better. Submit your suggestions through GitHub issues with the "enhancement" label.

Some ideas we're considering:

  • Altitude history tracking and graphs
  • Waypoint saving and navigation
  • Compass heading display
  • Export location data
  • Dark/light theme toggle
