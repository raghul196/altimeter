About Altimeter

What is Altimeter?

Altimeter is a web-based application that provides real-time altitude, location, and device orientation data using your device's built-in sensors and location services. Whether you're hiking, flying, or just curious about your elevation, Altimeter gives you accurate readings right in your browser.

Features

  • Real-time Altitude Tracking: Get your current elevation above sea level using GPS or Google's Elevation API
  • Inclinometer: Measure device pitch (forward/backward tilt) and roll (left/right tilt) using the DeviceOrientation API
  • Interactive Map: View your location on a map powered by Google Maps with real-time position updates
  • Coordinate Display: See precise latitude and longitude coordinates with accuracy information
  • Mobile-Optimized: Responsive design that works seamlessly on smartphones, tablets, and desktops

How It Works

Altitude Detection

The app first attempts to get altitude data from your device's GPS sensor using the Geolocation API. However, many mobile devices don't have barometric sensors or don't provide altitude through GPS. When altitude data is unavailable, the app automatically falls back to Google's Elevation API, which uses your coordinates to determine elevation based on terrain data.

Device Orientation

The inclinometer uses your device's gyroscope and accelerometer through the DeviceOrientation API. This provides pitch (tilting forward/backward) and roll (tilting left/right) measurements in degrees. On iOS 13 and later, you'll need to grant permission to access motion sensors.

Location Services

High-accuracy GPS positioning is used to track your location. The app continuously updates your position and displays it on an interactive Google Map. Coordinate precision depends on your device's GPS capabilities and environmental factors like satellite visibility.

Accuracy Notes

Altitude accuracy varies based on the data source. GPS altitude can have errors of 10-20 meters or more. Google's Elevation API typically provides better accuracy but depends on terrain data quality.

Location accuracy depends on GPS signal strength, which can be affected by buildings, trees, weather, and other environmental factors. Indoor use may result in reduced accuracy.

Orientation sensors are generally quite accurate but may need calibration. Different devices and browsers may report slightly different values.

Technology

Altimeter is built using modern web technologies:

  • HTML5 Geolocation API for GPS positioning
  • DeviceOrientation API for motion sensors
  • Google Maps JavaScript API for mapping
  • Google Elevation API for altitude fallback
  • Tailwind CSS for responsive styling
  • Pure JavaScript (no frameworks required)
